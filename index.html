To Do List

Este proyecto, es un Administrador de Tareas simple (To-Do List) está desarrollado en HTML, CSS y JavaScript, el objetivo principal es poder gestionar y organizar tareas de manera interactiva a través de la consola del navegador. El usuario puede agregar nuevas tareas, ver la lista completa de tareas pendientes y eliminar tareas completadas. Cada tarea está representada en un array que contiene su nombre e indice, lo que permite al usuario tener un control claro y directo sobre sus actividades. La interacción con el usuario se realiza mediante cuadros de diálogo (prompt, alert, console log y confirm).

El enfoque principal de esta aplicación es implementar la lógica de programación básica, utilizando arrays, funciones, bucles, condicionales y cuadros de diálogo en JavaScript. El menú de opciones se presenta al usuario en un bucle continuo, permitiendo seleccionar la acción deseada. Al estar orientado a la consola, este proyecto es ideal para reforzar conceptos fundamentales de JavaScript y preparar el terreno para futuras mejoras, como la integración de una interfaz gráfica con HTML y CSS.
